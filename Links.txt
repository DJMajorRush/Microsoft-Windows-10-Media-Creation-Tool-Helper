HowTo:
------


1) Rename file products_20171005.cab to products.cab,
2) Put file products.cab in the same folder where the file MediaCreationTool.exe
3) And run MediaCreationTool.exe /selfhost (or via ps: .\MediaCreationTool.exe /selfhost)
4) Save boot ISO..
5) You can tweak the parameters as needed: MediaCreationTool.exe /Selfhost /Eula Accept /Retail /MediaLangCode en-us /MediaArch x86 /MediaEdition Enterprise
6) Done.
